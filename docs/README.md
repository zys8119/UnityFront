# 快速入门

> 第一步, 下载项目 

```text
 【gitHub】仓库地址：https://github.com/zys8119/UnityFront
```

> 第二步，安装依赖

```bash
 npm install
```

> 第二步，运行项目

```bash
 npm run serve
```
